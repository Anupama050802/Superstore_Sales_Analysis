# 📊 Superstore Sales Analysis using SQL & Excel Dashboard

## 📌 Project Overview

This project analyzes the Superstore sales dataset using **SQLite** in Python for business analysis and **Microsoft Excel** for interactive dashboard creation. The objective is to extract meaningful business insights by solving real-world business problems using SQL queries and presenting the results through PivotTables, PivotCharts, KPIs, and an Excel dashboard.

---

## 🎯 Objectives

- Analyze overall sales, profit, orders, and customers.
- Identify top-performing categories, products, regions, and states.
- Study customer segments and shipping preferences.
- Analyze monthly sales trends.
- Evaluate the impact of discounts on profitability.
- Build an Excel dashboard to visualize key business insights.
